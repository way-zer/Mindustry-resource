# Mindustry 资源站前端

The source of the Mindustry resource site.  
技术栈: Vue3 Vite Typescript  
UI: element-plus

## 开发手册

IDE建议: IDEA(or Webstorm) 或 VSCode(with Volar)

```sh
pnpm install
pnpm run dev
```

### 附注

* vue的建议风格是 sfc+setup+hooks
* API接口,定义在store文件夹下,如需要存储状态,请建立store

> 所有后端Api接口禁止它用,如愿参与协作,请与WayZer联系
